# PythonCompiler
A simple Compiler for C language using python


---

# Help
Compiler for C using python

Options:

        -h, --h         show help
        
        -s file         import the source file, required!
        
        -l              lexer
        
        -p              parser
        
Examples:

    python *.py -h
    python *.py -s source.c -l
    python *.py -s source.c -p
